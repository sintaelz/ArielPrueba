
CTC example
-----------

This directory contains following files

   readme.txt      -- This file
   fulldemo.sh     -- Demo-script
   makefile        -- Makefile to make the example

   reverse.h
   reverse.c       -- Reverse module
   strutil.h
   strutil.c       -- String utility module
   main.c          -- Main program

This example demonstrates testing of multiple libraries in same
executable.

Following targets are defined for the makefile

   make all      Build the example (default target), demo
   make clean    Delete all produces files
