
CTC example
-----------

This directory contains following files

   readme.txt      -- This file
   makefile        -- Makefile to make the example
   fulldemo.sh     -- Demo-script

   cstack.h        -- \
   cstack.cc       --  \ The tested classes
   revstack.h      --  /
   revstack.cc     -- /
   stacktst.cc     -- A test main function

This example demonstrates testing of a (very) simple program with CTC++.

Following targets are defined for the makefile

   make all      Build the example (default target), stacktest
   make clean    Remove files produced with this makefile

To see a demo of the CTC++-usage with stacktest-program run the script
fulldemo.sh.
